# **App Name**: DesignCraft Studio

## Core Features:

- Product Catalog & Selection: Browse and select from various printable product categories such as business cards, stickers, flyers, brochures, and invitations. Each product will have configurable dimensions and basic templates.
- Basic Web-based Design Editor: An intuitive, canvas-based editor allowing users to add, position, and resize text elements, basic shapes, and upload their own images to the selected product template.
- Real-time Design Preview: Instantly visualize how the design looks on the selected product as changes are made, offering both front and (if applicable) back views.
- AI Design Assistant: A tool that suggests layout ideas, font pairings, or color palettes based on the selected product type and user-defined preferences (e.g., 'minimalist', 'vibrant').
- Save & Load User Designs: Users can save their in-progress designs to return to later, and load previously saved designs for further editing or ordering.
- Design Export & Order Summary: Generate a printable PDF proof of the finalized design. A simple order summary will be displayed, outlining the product details and mock pricing (without actual payment processing in the MVP).

## Style Guidelines:

- Primary color: A sophisticated, deep blue (#2E58B2) to convey professionalism and trust, balanced with creativity.
- Background color: A very light, cool grey (#F3F5FA) for a clean, expansive canvas that highlights design elements.
- Accent color: A vibrant, clear cyan (#2CCBFF) to draw attention to interactive elements, CTAs, and create visual interest.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring excellent readability across the application interface.
- Utilize a consistent set of minimalist, vector-based line icons to represent design tools, features, and product types. Focus on clarity and ease of recognition.
- The interface will feature a central, prominent design canvas, flanked by clearly defined, collapsible panels for tools, layers, and properties, optimized for a responsive desktop experience.
- Implement subtle, non-intrusive transition animations for panel expansions/collapses and state changes, providing clear feedback without distracting from the creative process.
